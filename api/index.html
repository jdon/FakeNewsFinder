Check fake news by URI

Format: JSON
Location: /url/[URI]
Example: http://fake.jdon.uk/url/http%3A%2F%2Fwww.independent.co.uk%2Fnews%2Fuk%2Fpolitics%2Fbrexit-theresa-may-article-50-negotiations-security-threat-latest-leave-eu-european-union-a7659466.html

Sample Results

	
	{
		"domain": "www.independent.co.uk",
		"title": "Theresa May forced into her first humiliating climbdown since triggering Brexit",
		"author": "Joe Watts Political Editor",
		"domainList": {		//results of test by domain list
			"fake": false,
			"notes": null
		}
	}

Sample Domain List Result

	
	"domainList": {
		"fake": true,
		"notes": {	//if not fake, this value will be null
			"type1": "fake",	//up to 3 types of fake site are given
			"type2": "rumor",
			"type3": "clickbait",
			"notes": "pretty well known tabloid/fake news source"	//any extra notes on the site
		}
	}
